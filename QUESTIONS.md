# Questions I Would Ask

1. Should clients be able to request jobs themselves, or should admins create all jobs?
2. Do technicians need availability calendars, working hours, or travel-time constraints?
3. Which status changes should notify clients versus only internal users?
4. Should client-visible notes be separate from internal notes for compliance reasons?
5. Are jobs ever deleted, or should completed and cancelled jobs always remain archived?
6. Is email notification required in v1, or are in-app notifications enough?
7. Should admins be able to create new users in the app, or is seeded/invited access acceptable for the first release?
